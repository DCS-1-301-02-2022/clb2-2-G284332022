# 情報工学科の紹介

### コンピュータの基礎から、高度なプログラミングまで。情報化社会をさらに進化させる、IT業界の新たなリーダーを育成。
**２年次から3つのコースから選択**
1. **システムエンジニアリングコース**
1. **インターネット＆セキュリティコース**
1. **知能情報コース**
- *システムエンジニアリングコース*
   - プログラミングによって時代の求める新しい情報システムを作る。
- *インターネット＆セキュリティコース*
  - 安全で安心できる持続的な情報社会を支えるネットワークとセキュリティを学ぶ。
- *知能情報コース*
  - 「モノ」と「コト」の未来へAIの先を拓く。
